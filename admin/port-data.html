Port Data Management

Guidance: Understanding Port Data

Accurate port data is the foundation of every route calculation. This information determines the start, end, and intermediate points of every voyage, and a realistic estimate of potential delays.

Why This Data is Critical for the Algorithm
  • Latitude & Longitude: These are the exact geographic coordinates of a port. The algorithm uses them as the core waypoints to calculate distances, plot routes on the map, and determine the most efficient path. Without them, distance and fuel calculations are impossible.
  • Congestion Index: This acts as a "delay penalty" for the algorithm. A high index tells the system that a port is frequently busy, adding a calculated waiting time to the total voyage duration. This allows the algorithm to make smarter decisions, like choosing a slightly longer route to a faster port, which may save time and money overall.
How to Find Accurate Data
  • For Latitude & Longitude (Official Source):
    The industry standard for location data is the UN/LOCODE list maintained by the United Nations. This provides a unique code and coordinates for thousands of transport locations worldwide.
    • Source: UN/LOCODE Code List
    • How-to: Search for your port on the official list. The coordinates are provided in the "Coordinates" column.
  • For Congestion Index (Live Data Source):
    Congestion is dynamic. While internal company data is valuable, specialized maritime analytics tools provide the best near-real-time data by analyzing global vessel traffic.
    • Source: SeaDex.ai Port Congestion Tool
    • How-to: Use this free tool to view the current and historical congestion levels for major ports. Use this data to enter a realistic index (e.g., a percentage or a 1-100 score).
Pro-Tip for Operators: For the most precise voyage calculations, use the Latitude and Longitude of the port's main channel entrance buoy (also called the "sea buoy" or "fairway buoy"). This represents the true start and end point of the open-ocean segment of a voyage.
