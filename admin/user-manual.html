Multi-Port Optimization Benefits

The TSP-based route optimization provides:

  • Optimal Sequencing: Automatically finds the best order to visit all ports
  • Fuel Savings: Reduces total distance traveled and fuel consumed
  • CO₂ Reduction: Lower emissions through efficient routing
  • Time Efficiency: Minimizes overall voyage duration
  • Visual Comparison: Clear side-by-side comparison of standard vs. optimized routes
Pro Tip

The map shows marine zones including EEZ boundaries and territorial seas. Use this information to understand maritime jurisdictions along your route. The system also displays weather data like wind patterns and wave heights to help plan safer voyages.

How It Works

The system uses a Genetic Algorithm to solve the Traveling Salesman Problem (TSP). It compares your entered sequence (Standard Route) against the mathematically optimized sequence (Optimized Route) and shows the improvements in a comprehensive comparison table.

Automatic Calculation Features

The system automatically calculates fuel consumption using the Simonsen et al. (2018) methodology:

  • Speed-Ratio Method: Uses actual MCR and speed ratio (not Admiralty Formula)
  • Weather & Fouling Factors: τw = 1.15, τf = 1.09 (from paper)
  • Load-Dependent SFC: Accounts for engine efficiency changes with load
  • Hotel Load: Size-based auxiliary power estimation (3-25 MW)
  • Academic Validation: Verified against MS Finnmarken, MS Artania, Norwegian Star
Pro Tip

The Cruise Ship Database displays 9 columns: Ship Name, Operator, Gross Tonnage, Propulsion Power, Cruising Speed, Max Speed, Length, Year Built, and Fuel Type. Click the chart icon to visualize each ship's fuel consumption curve with detailed breakdowns of consumption at different speeds.

Data Source Recommendation

For accurate specifications, use data from CruiseMapper or official ship documentation. All critical fields (tonnage, power, speeds, dimensions) are required for the automatic fuel curve calculation to work properly. The system uses these values with marine engineering formulas to generate realistic fuel consumption estimates.

Calculation Component Formula (Simonsen et al. 2018) Purpose
Power at Speed (Eq. 12) P = MCR × (V/Vref)³ × 1.15 × 1.09 Calculates propulsion power using speed ratio
Load Factor (Eq. 14) L = P / MCR Determines engine loading percentage
SFC (Eq. 17) SFC = 195 × [1.28 - 0.71L + 0.455L²] Load-dependent fuel consumption rate
Hotel Load 15-25 MW for mega ships Auxiliary power (HVAC, lighting, services)
Fuel Consumption (Eq. 19) F = (P + Hotel) × SFC / 1000 Final fuel consumption in tons/hour

The Port Weather system includes automated safety assessments based on:

Frequently Asked Questions

Find answers to common questions about NauticalFlow. If you can't find what you're looking for, please contact our support team.

How accurate are the fuel consumption calculations?

Our calculations are based on the fuel consumption curves you provide for each vessel, combined with real-time weather data and port congestion information. Accuracy typically ranges from 92-98% when vessel data is properly calibrated.

Can I optimize routes with multiple destination ports?

Yes! Our multi-port optimization feature uses advanced algorithms (Genetic Algorithm for TSP) to find the most efficient sequence of ports to visit, minimizing total fuel consumption and travel time.

What weather data sources does NauticalFlow use?

We integrate with Open-Meteo and Marine Weather API to provide real-time weather forecasts, wave conditions, wind patterns, and cyclone tracking for accurate route planning.

How do I update my vessel's fuel consumption curve?

Navigate to the Vessel Data page, find your vessel, and click the edit button. You can update the fuel consumption curve data points, which will be used in all future route optimizations.

Can I export my optimization history?

Yes! You can export your data in multiple formats including PDF reports for presentations and CSV files for further analysis in spreadsheet applications.

Is my data secure?

Yes, we take data security seriously. All data is encrypted in transit and at rest. We use industry-standard authentication protocols and regularly backup all information. Your vessel and route data remains private and confidential.

What happens if I delete a vessel that's used in saved optimizations?

Historical optimization records will retain the vessel name, but you won't be able to use that vessel for new optimizations. We recommend archiving vessels instead of deleting them if they have historical data.

How often is port and weather data updated?

Port data is updated manually as needed. Weather data is refreshed every 6 hours from our API sources, ensuring you always have current information for route planning decisions.
