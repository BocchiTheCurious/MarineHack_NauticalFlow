Guidance: How Ship Profiles Work

This page is where you manage your fleet's performance profiles. An accurate profile is essential for the route optimization algorithm.

When you add a ship, you only need to provide its basic details. The system will then use the ship's Gross Tonnage (GT) to automatically select the best performance template and generate a realistic Fuel Consumption Curve for the algorithm.

Why This Data is So Important

A ship's fuel consumption increases exponentially with its speed. The auto-generated curve provides the algorithm with the data it needs to analyze thousands of route variations and find the perfect balance between speed, safety, and COâ‚‚ emissions.

System Templates & Methodology

The system uses the following archetypes to generate a credible baseline fuel curve. These are based on an analysis of real-world ship classes and standard naval architecture models.

  • Legacy Mid-Size Ship: For GT between 70,000 - 90,000
  • Mid-Size Diesel-Electric Ship: For GT between 90,001 - 130,000
  • Large Modern Ship: For GT above 130,001
Our templates are generated using a scientific approach. We use high-level specifications from public sources for different ship classes and apply standard naval architecture models (like the Propeller Law) and academic research on cruise ship energy use to create a realistic and credible baseline fuel curve. This provides a much more accurate starting point than manual guesswork.
